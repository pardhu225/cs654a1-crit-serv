* Requires Node.JS
* Run
    npm install
  in the root folder of the project to install all the dependencies
* Run 
    npm run start [VM1/VM2]
  to start the respective server.
* Add the server IP and validation details to nodes.json file in VM3 and restart the server

* NOTE: You can send GET /set-availability?av=[true/false] to set the availability of this server
